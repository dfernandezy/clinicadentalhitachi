Accesibilidad

Compromiso con la accesibilidad digital

Última actualización: 30/01/2025

Declaración de Accesibilidad

En Centro Dental Hitachi nos comprometemos a hacer que nuestro sitio web sea accesible para todos los usuarios, independientemente de sus capacidades o tecnología utilizada.

Nuestro Compromiso

Nos esforzamos por alcanzar el nivel AA de las Pautas de Accesibilidad para el Contenido Web (WCAG) 2.1, que incluye:

  • Texto alternativo para imágenes
  • Estructura clara y navegación coherente
  • Contraste de color adecuado
  • Compatibilidad con tecnologías de asistencia
  • Acceso mediante teclado

Características de Accesibilidad

Nuestro sitio web incluye las siguientes características de accesibilidad:

  • Navegación consistente en todas las páginas
  • Enlaces descriptivos y únicos
  • Textos claros y legibles
  • Formularios etiquetados correctamente
  • Contenido responsive adaptable a diferentes dispositivos

Tecnologías de Asistencia

Nuestro sitio web es compatible con las siguientes tecnologías de asistencia:

  • Lectores de pantalla
  • Software de reconocimiento de voz
  • Magnificadores de pantalla
  • Teclados alternativos

Limitaciones Conocidas

Trabajamos continuamente para mejorar la accesibilidad de nuestro sitio. Si encuentra alguna barrera de accesibilidad, por favor háganoslo saber.
